Test the JSON to TypeScript converter keyboard shortcut

Instructions:
  1. Select any JSON sample below (click and drag to select)
  2. Press the keyboard shortcut:
    • Mac: Control + Command + C
    • Windows/Linux: Control + Shift + C
  3. Check for a notification at the top-right of the page
  4. The TypeScript type should be copied to your clipboard

JSON Sample 1: Simple User Object

{ "name": "John Doe", "email": "john@example.com", "age": 30, "isActive": true }

JSON Sample 2: Nested Object

{ "user": { "id": 123, "profile": { "firstName": "Jane", "lastName": "Smith", "avatar": "https://example.com/avatar.jpg" }, "preferences": { "theme": "dark", "notifications": true } } }

JSON Sample 3: Array of Objects

{ "products": [ { "id": 1, "name": "Laptop", "price": 999.99, "inStock": true, "tags": ["electronics", "computers"] } ], "total": 1 }

üêõ Debugging Steps if Not Working:

  1. Open Chrome DevTools: Press F12 or right-click ‚Üí Inspect
  2. Go to Console tab and look for these messages:
    • "Content script loaded on: ..." - Shows content script is injected
    • "Keyboard shortcut detected in content script!" - Shows shortcut was pressed
    • "Sending message to background script..." - Shows message is being sent
    • "Received response from background:" - Shows response received
  3. Go to Extension's Service Worker:
    • Click the puzzle icon (Extensions) in Chrome toolbar
    • Click "Manage extensions"
    • Find "JSON to TypeScript Converter"
    • Click "service worker" link to open its console
    • Look for: "Message received in background:" and other debug logs
  4. Check for errors: Any red error messages in either console
  5. Verify extension is enabled: chrome://extensions/
  6. Try reloading the extension: Click the reload icon on chrome://extensions/
